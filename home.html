About This Website

Alexlais.com is an AngularJS single page application using ngRoute with HTML5 Mode enabled and loads data thru a ngRepeat with a routeParams filter based on the current URL.

Hosted by GitHub
This Angular application is currently hosted by GitHub. The production repository is github.com/alexlais/dotcom.
